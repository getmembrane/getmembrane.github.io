Our service is built with proven technology in a robust, reliable and scalable fashion. When a user is about to comment on your site a request is sent to our API. The json-response contains information whether the user is a known hater or not, allowing your site to take action.

Using crowdscreen is easy. If your running on any of the large platforms, just install the corresponding plugin and you are ready to go. If your platform need something custom we have a perfectly documented API complete with examples for the most common languages (hey we're developers too you know!).
